## Big Os of Data Stucture
A data “structure” is like a container that stores data in a specific, organized layout. 

### stacks:
- insertion - O(1)
- removal - O(1)
- searching - O(N)
- Access - O(N)

### queus:
- insertion - O(1)
- removal - O(1)
- searching - O(N)
- Access - O(N)


### BST
- Insertion - O(log n)
- Searching - O(log n)
- Not guaranteed
